## 方案一：「赛博朋克像素办公室」

### Design Movement
赛博朋克 × 像素艺术 — 霓虹灯光、深色背景、高科技低生活的量化交易所

### Core Principles
1. 深色基调搭配霓虹高亮，营造"高频交易暗室"氛围
2. 数据流动感 — 所有 UI 元素暗示数据在流动
3. 终端美学 — 弹窗和面板模拟终端/命令行风格
4. 信息密度高但层次分明

### Color Philosophy
- 主背景：深蓝黑 #0a0e1a
- 主强调色：电光蓝 #00d4ff、霓虹绿 #39ff14
- 警告色：霓虹粉 #ff006e
- 文字：冷白 #e0e6ed
- 情感意图：紧张、专业、前沿科技感

### Layout Paradigm
全屏像素办公室场景为主视图，底部固定 HUD 状态栏（积分、日期、公司等级），右侧可展开的任务面板。点击研究员弹出浮动状态卡片。

### Signature Elements
1. 像素化的数据流粒子在办公室背景中流动
2. 研究员头顶的状态气泡（进度条、情绪图标）
3. 霓虹边框的弹窗面板

### Interaction Philosophy
点击即反馈 — 每次交互都有像素化的闪光和音效提示感（视觉模拟）。拖拽研究员到工位分配任务。

### Animation
- 研究员像素角色 idle 动画（2-3帧循环）
- 数据流粒子持续流动
- 弹窗从像素点展开到完整面板
- 积分变化时数字滚动效果

### Typography System
- 标题：Press Start 2P（经典像素字体）
- 正文：Silkscreen（清晰像素字体）
- 数据：VT323（等宽终端字体）

## 方案二：「复古 Game Boy 办公模拟器」

### Design Movement
Game Boy 复古像素 × 经营模拟 — 受《口袋妖怪》和《牧场物语》启发的温暖像素风

### Core Principles
1. 有限调色板（4-5色）营造统一复古感
2. 温馨可爱的角色设计，降低量化的"冷硬"门槛
3. 清晰的网格系统，一切对齐到像素网格
4. 游戏感优先 — 每个操作都像在玩游戏

### Color Philosophy
- 主色调：暖米色 #f5e6c8 背景
- 主强调：深棕 #5a3921、橄榄绿 #6b8e4e
- 高亮：琥珀金 #d4a017
- 文字：深咖啡 #2d1b0e
- 情感意图：亲切、怀旧、有趣、降低学习焦虑

### Layout Paradigm
上方为像素办公室俯视图（可滚动），下方为固定的游戏控制面板（类似 GBA 游戏的下屏）。左侧迷你地图显示办公室全貌。

### Signature Elements
1. Game Boy 风格的圆角窗口边框（双线像素边框）
2. 角色对话气泡用像素化漫画风
3. 金币/积分用像素动画硬币表示

### Interaction Philosophy
按钮式交互 — 所有操作通过明确的像素按钮完成，模拟掌机体验。菜单层级清晰，A键确认B键返回的逻辑。

### Animation
- 角色 4 方向行走动画
- 金币收集弹跳动画
- 窗口打开/关闭的缩放动画
- 进度条填充时的像素化闪烁

### Typography System
- 全局：Press Start 2P
- 数据面板：Silkscreen
- 保持 8px 基准网格对齐

## 方案三：「现代像素 × 等距办公室」

### Design Movement
现代等距像素艺术 — 受 Gather Town、Habitica 启发，融合现代 UI 设计与像素美学

### Core Principles
1. 像素办公室场景与现代 UI 面板共存 — 游戏世界用像素，数据面板用现代设计
2. 双层视觉系统：底层像素世界 + 上层半透明现代面板
3. 信息可视化优先 — 量化数据用现代图表，游戏元素用像素风
4. 渐进式复杂度 — 界面随公司成长逐步解锁

### Color Philosophy
- 办公室场景：柔和的蓝灰色调 #c8d6e5 地板、#dfe6ed 墙壁
- UI 面板：深色毛玻璃 rgba(15,23,42,0.85)
- 主强调：靛蓝 #6366f1、翡翠绿 #10b981（盈利）、珊瑚红 #ef4444（亏损）
- 积分金色：#fbbf24
- 情感意图：专业但有趣、现代但怀旧、数据驱动但游戏化

### Layout Paradigm
全屏等距像素办公室为核心视图。顶部固定导航栏（公司名、积分、日期）。点击研究员/设备弹出右侧滑入面板（现代 UI 风格）。底部快捷操作栏。整体类似 RTS 游戏的 HUD 布局。

### Signature Elements
1. 等距视角的像素办公室，家具和设备有微动画
2. 研究员头顶的状态指示器（小图标 + 进度环）
3. 面板使用毛玻璃 + 像素边框的混搭风格，连接两个视觉世界

### Interaction Philosophy
探索式交互 — 鼠标悬停显示 tooltip，点击打开详情面板。办公室中的每个物件都可交互。拖拽分配任务。双击快速操作。面板支持最小化/固定。

### Animation
- 研究员 idle 动画（打字、思考、喝咖啡）
- 电脑屏幕闪烁效果
- 面板滑入/滑出带弹性动画
- 策略上线时的"火箭发射"像素动画
- 盈利时金币飘落效果
- 数据图表实时更新动画

### Typography System
- 游戏标题/像素元素：Press Start 2P
- UI 面板标题：Space Grotesk（现代几何无衬线）
- 面板正文/数据：JetBrains Mono（等宽，适合数据展示）
- 中文回退：Noto Sans SC

## 最终选择：方案三「现代像素 × 等距办公室」

选择理由：
1. 双层视觉系统完美解决了"像素游戏感"与"量化数据专业展示"的矛盾
2. 等距视角更接近参考图片的风格
3. 现代 UI 面板让数据展示更清晰，不会因为全像素化而牺牲可读性
4. 渐进式复杂度设计符合游戏化学习的核心理念
